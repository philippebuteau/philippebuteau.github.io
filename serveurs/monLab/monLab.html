Mon Laboratoire

Infrastructure virtuelle simulant une petite entreprise — VMware Workstation Pro.

Contexte

Cette page documente un projet personnel que j'utilise pour approfondir la matière vue dans le cadre de mes cours.

Je réalise ce projet dans un environnement entièrement virtualisé, en utilisant VMware Workstation Pro comme hyperviseur.

Mon approche est de simuler une infrastructure de petite entreprise complète.

Mandat
  • Implémenter un serveur DHCP et DNS sous Linux
  • Installer un serveur de stockage avec RAID
  • Installer un pare-feu (porte vers internet)
  • Configurer un hyperviseur en cluster
  • Configurer un accès à distance (VPN / thin clients)
  • Plan d'adressage avec segmentation VLSM
  • Utiliser des VLANs dans le plan d'adressage
  • Configurer un contrôleur de domaine
  • Configurer un serveur de messagerie
  • Configurer un serveur web
