Outil d'affichage de composantes / wallpaper dynamique

Contexte

Dans le contexte de mes cours, je dois régulièrement installer de nouvelles machines virtuelles. Chaque nouvelle machine représente un environement de travail différent. Je dois souvent vérifier certains paramètres sur ces machines, comme le système d'opération, les addresses ip, le stockage, le matériel installé et l'utilisation des ressources. J’ai donc décidé de créer un script permettant de voir immédiatement sur quoi je travaille grâce à l’arrière‑plan, directement sur le bureau.

Mandat

  • Créer un objet qui catalogue les composantes de l’ordinateur courant
  • Créer une image via un script pouvant être utilisée comme arrière‑plan
  • Formater les données de l’objet en texte
  • Assurer un placement adaptatif du texte sur l’image
  • Configurer un cron job pour mettre l’image à jour chaque minute
  • Faciliter l’installation sur une nouvelle machine
