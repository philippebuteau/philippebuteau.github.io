Outil de configuration de serveur linux

Contexte

Toujours dans le cadre de mes cours, il est régulièrement nécéssaire de créer de nouvelles installations de systèmes d'opération linux, sur lesquels nous configurons des services réseaux. Pour chaque nouvelle machine, la même configuration initiale est nécéssaire. Il faut configurer le réseau pour que les machines communiquent entre elles, il faut configurer le pare-feu, selinux, installer les paquets, configurer le nom de la machine. Après avoir fait tout ça, le travail abordé dans le cours concerné n'est même pas commencé, et il sera tout à recommencer lors du prochain laboratoire.

Certains services se cadrent aussi dans le même problème. Je pense au services DHCP et DNS particulièrement. Ils sont souvent à la base de d'autres procédures. C'est donc pour accélérer ces configurations multiple que je m'attaque à ce projet.

Sur linux, tout se configure en texte dans des fichiers spécifiques. Mon objectif est de construire une structure paramétrable pour rendre tous ces services opérationnels en demandant à l'administrateur de répondre à quelques question dans un terminal. La réeussite de cet objectif me permettra de sauver un temps fou pour me concentrer sur la nouvelle matière, et non sur ce que j'ai déjà fait des dizaines de fois.

Cet outil sera itératif et pourra être mis à jour régulièrement pour inclure mes nouveaux apprentissages.

Mandat

  • Créer un outil basé sur terminal permettant de configurer rapidement une nouvelle machine avec :
    • Configuration Réseau
    • Configuration du pare-feu
    • Configuration de SELINUX
    • Configuration d'un serveur DHCP
    • Configuration d'un serveur DNS
    • Configuration d'un serveur NFS
    • Configuration d'un serveur LDAP
    • Configuration d'un serveur Web
    • Formattage d'un nouveau disque dur
    • Configuration de clés SSH
