Outil Bash interactif permettant de configurer rapidement plusieurs types d'interfaces réseau

Contexte

Cette page documente un projet qui visait d'abord à créer un script pour configurer TCP/IP sur une interface rapidement. Ce script m'a mené vers l'exploration d'autres types de configurations, tels que les VLANS et les interfaces de types 'bond' et 'team'

Vous trouverez donc ici un outil pour faciliter la création de ces interfaces sous Linux

Mandat
  • Outil terminal pour configurer rapidement plusieurs types d'interfaces :
    • Configuration ipv4
    • Configuration de la passerelle par défaut
    • Configuration des addresses des serveurs DNS
    • Configuration de la méthode d'addressage (Statique ou Automatique)
    • Configuration d'un profil d'interface ethernet
    • Configuration d'une interface en VLAN
    • Configuration de plusieurs interfaces en 'bond'
    • Configuration de plusieurs interfaces en 'team'
    • Suppression du profil d'une interface
