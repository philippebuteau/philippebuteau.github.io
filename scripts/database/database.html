Outil de configuration de serveur linux

Contexte

Cette outil fait partie du contenu d'un cours en gestion de bases de données relationnelles. L'objectif est d'automatiser certains processus avec un script bash.

Les processus concernés sont par l'autmatisation sont principalement en lien avec les sauvegardes de données. Ces dernières doivent se programer facilement pour etre sauvegarder reguilierement. De plus, les sauvegardes doivent etre configurable pour se diviser en base de donnees et en table au besoin, tout en permettant l'usage d'options.

Les sauvegardes vont donc s'accumuler et prendre de l'espace disque inutilement, il faut aussi automatiser le ménage des données. Se ménage doit etre organisé pour conserver un certain nombre de sauvegardes pour chaque type de sauvegardes qui existent sur le disque. Les sauvegardes doivent donc etre nommees de facon coherente et automatiquement pour eviter les erreurs.

Le script permettra aussi la creation d'utilisateurs, la gestion des permissions, le chiffrement des données avec un mot de passe et plus.
