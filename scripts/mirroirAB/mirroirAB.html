Outil de mise en miroir
de disque externe

Script PowerShell / Bash pour synchroniser des dossiers entre deux emplacements — maison et école.

PowerShellBashJSONWindowsLinuxAdministration système
Contexte

L'une des choses qui m'a frappé lorsque j'ai commencé mes cours est un processus informatique qui efface toutes les données des étudiants quotidiennement. En conséquence, je devais régulièrement transporter mes fichiers entre l'école et la maison via un disque externe.

À mesure que le nombre de projets augmentait, ce processus manuel devenait long et sujet aux erreurs. J'ai donc créé un outil pour automatiser la synchronisation.

Note : Cette solution a éventuellement été remplacée par une connexion VPN via cellulaire avec RustDesk.

Mandat
  • Accès aux données les plus récentes aux deux endroits via le disque externe
  • Capacité de définir des dossiers favoris pour éviter les duplications
  • Configuration des favoris enregistrable
  • Liste noire pour omettre certains éléments
  • Processus rapide et simple
  • Script central réutilisable avec paires A–B configurables
  • Compatible Windows et Linux
Résumé de la solution
  • Analyse le contenu d'un fichier JSON pour extraire les favoris en paires source / destination
  • Collecte les données des fichiers des deux répertoires et crée un inventaire
  • Affiche les fichiers manquants et modifiés à l'utilisateur
  • Chaque fichier reçoit un statut et une action, exécutée après confirmation
  • Met à jour les fichiers désuets et copie les fichiers manquants — minimum d'opérations
  • Inclut une liste noire pour exclure des emplacements
  • Offre un script secondaire pour modifier les favoris et la liste noire
  • Mode automatique disponible

# Description : 
# À partir du concept du script exemple Explorateur de fichiers,
# le script suivant est un outil pour faire un backup "intelligent" entre
# un dossier racine A et un dossier racine A'.
 
# Compare deux répertoires et enregistre les informations de chaque emplacement dans un objet.

# Une fois cette analyse complète, le processus suivant s'exécute :
#    -Les fichiers uniquement dans A sont copiés dans A',
#    -Les fichiers uniquement dans A' sont copiés dans A, 
#    -Les fichiers modifiés dans A sont mis à jour dans A',
#    -Les fichiers modifiés dans A' sont mis à jour dans A,

# Les fichiers sont donc maintenus à jour dans les deux répertoires.

# Le script est concu pour iterer sur une liste (favoris.json) de repertoires A-A'
# Une option peut-etre selectionne pour modifier les favoris

# Pour definir une nouvelle paire de favoris, une paire de racines (racines.json) est utilisee;
#	Exemple:
# 		racineA: /A
#		racineB: /B/C
#		nouveau favoris A: /A/a/b/c (entree de l'utilisateur)
#		nouveau favoris A': /B/C/a/b/c (genere automatiquement en remplacant racineA par racineB dans le nouveau favoris A)

# Une blacklist (blacklist.json) est utilisee pour specifier des fichiers a omettre ou supprimer

# FONCTION pour analyser le contenu de d'un emplacement et retourner un objet personnalisé
#  Représentant les informations de chaque emplacement, le statut par rapport au backup et
#  l'action a prendre.
#============================================================================================
function Get-mirrorImage {
    param (
        [int]$key,
        [string]$path,
        [hashtable]$inventaires,
        [hashtable]$racines,
        [array]$Statuts,
        [array]$Actions,
        [int]$direction,
        [bool]$birectionnel
    )
    foreach ($j in $inventaires[$key].GetEnumerator()) {
    
        # Vérification de l'existence d'un backup de l'objet courant 
        $reflection = $j.FullName.Replace($racines[$key], $racines[$key + $direction])
        $testItem = Test-Path $reflection         
        if ($testItem -eq $True -and $j.FullName) {
                
            # Stockage de l'objet représentant le backup de l'objet courant    
            $reflection_objet = Get-Item -Path $reflection -ErrorAction SilentlyContinue

            # Si le backup est plus récent que l'objet courant
            if ($reflection_objet.LastWriteTime -gt $j.LastWriteTime) {
                $Statut = $Statuts[2]
                $Action = $Actions[3]
            # Si l'objet courant est plus récent que le backup
            } elseif ($reflection_objet.LastWriteTime -lt $j.LastWriteTime) {
                $Statut = $Statuts[5]
                $Action = $Actions[4]
            # Si les deux objets ont la même date de modification
            } else {
                $Statut = $Statuts[4]
                $Action = $Actions[4]                    
            }
	    # Si la reflection de l'objet courant est sur la liste noire
	    foreach ($item in $fullBlacklist){
		if ($reflection_object){
	            if ($reflection_objet.FullName.StartsWith($item)){
                        $Statut = $Statuts[7]
                        $Action = $Actions[4]
	            }
		}
	    }    
        # Si le backup de l'objet courant n'existe pas
        } elseif ($testItem -eq $False) {
            $Statut = $Statuts[0]
            $Action = $Actions[0]
        }
        # Si l'objet courant est un répertoire
        if ($j.PSIsContainer){
            $Statut = $Statuts[6]
            $Action = $Actions[4]          
        }
	# Si l'objet courant est sur la liste noire
	foreach ($item in $fullBlacklist){
	    if ($j.FullName.StartsWith($item)){
                $Statut = $Statuts[7]
                $Action = $Actions[4]
	    }
	}    
               
        $infos[$j.FullName] = [PSCustomObject]@{
            FullNameA = $j.FullName
            FullNameB = $reflection
            LengthA = "$([math]::Round($j.Length / 1KB, 2)) KB"
            LengthB = if ($testItem) { "$([math]::Round($reflection_objet.Length / 1KB, 2)) KB" } else { "N/a" }
            ModA = $j.LastWriteTime
            ModB = if ($testItem) { $reflection_objet.LastWriteTime } else { "        N/a   " }
            Statut = $Statut
            Action = $Action
        }
    }
    return $infos
}

# FONCTION pour créer les répertoires manquants
#============================================================================================
function Create-Parents {
    param (
        [string]$cheminFichier
    )
    $dossierParent = Split-Path $cheminFichier -Parent
    if (-not (Test-Path $dossierParent) -and $dossierParent) {
        Write-Warning "L'emplacement $($dossierParent) était manquant, il a donc été créé."
        New-Item -ItemType Directory -Path $dossierParent -Force | Out-Null
    }
}

#============================================================================================
# Création des racines manquantes
#============================================================================================
foreach ($i in $racines.GetEnumerator()) {
    Create-Parents -cheminFichier $i.Value
}

#============================================================================================
# Bloc de script pour itérer au travers des paires d'emplacements racine en executant la
# fonction Get-mirrorImage
#============================================================================================
foreach ($i in $racines.GetEnumerator()) {

    # Dans chaque racine, stockage des objets enfants de façon récursive
    
    $inventaires[$i.Key] = @(Get-ChildItem -Path $i.Value -Recurse -Force)

    # Si l'itération courante est un emplacement source
    if ($i.Key % 2 -eq 0) {      
        # Pour chaque objet de l'inventaire de la racine courante
        $infos[$i.Key] = Get-mirrorImage -key $i.Key -path $i.Value -inventaires $inventaires -racines $racines -Statuts $Statuts -Actions $Actions -direction 1
    } else {
        # Pour chaque objet de l'inventaire de la racine courante
        $infos[$i.Key] = Get-mirrorImage -key $i.Key -path $i.Value -inventaires $inventaires -racines $racines -Statuts $Statuts -Actions $Actions -direction -1

}

if ((-not $confirmation) -or ($confirmation -in "y", "Y") -and ($dryrun -eq $False)) {

        #============================================================================================
        # Mirroir des emplacements A et A'
        #============================================================================================
        foreach ($i in $infos[$item.key].GetEnumerator()) {

            if (($i.Value.Statut -eq $statuts[7]) -and ($dodelete -eq $True)){

		Write-Host "$($i.Value.Action) -> $($i.Value.FullNameA) -> $($i.Value.FullNameB)"
		Copy-Item -Path $i.Value.FullNameA -Destination $i.Value.FullNameB -Force -ErrorAction SilentlyContinue
                "ACTION: $($i.Value.Action) -> $($i.Value.FullNameA) -> $($i.Value.FullNameB)" >> $log
		                
		# Update les Statuts et Actions de l'item pour ne pas copier l'item de nouveau lors de l'image mirroir
                $i.Value.Action = $Actions[4]
                $i.Value.Statut = $Statuts[4]
        
            } elseif ($i.Value.Action -eq $actions[1] -or $i.Value.Action -eq $actions[3]) {

                Create-Parents -cheminFichier $i.Value.FullNameA
                Write-Host "$($i.Value.Action) -> $($i.Value.FullNameB) -> $($i.Value.FullNameA)"
                Copy-Item -Path $i.Value.FullNameB -Destination $i.Value.FullNameA -Force
                "ACTION: $($i.Value.Action) -> $($i.Value.FullNameB) -> $($i.Value.FullNameA)" >> $log

                # Update les Statuts et Actions de l'item
                $i.Value.Action = $Actions[4]
                $i.Value.Statut = $Statuts[4]
            }
        }
    }
    Write-Host "`nOpération Terminée." -ForegroundColor Green
} else {
	if ($dryrun -eq $True){
		Read-Host "`nDry Run completed. Press Enter to quit"
	} else {
		Write-Host "`nOpération annulée par l'utilisateur; Aucune action n'a été prise." -ForegroundColor Red
